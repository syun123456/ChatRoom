# ChatRoom (交友聊天室練習網站)

使用工具：Java、Spring Boot、Mybatis、Redis、WebSocket

網站功能：帳號登入驗證、多人聊天室、好友私訊
  

一、帳號登入驗證
-------

* 未登入攔截

* 帳號密碼驗證

* 帳號新增、修改、刪除

* 設置個人頭貼

二、多人聊天室 (透過Ajax實現聊天功能)
-------

* 建立聊天室

* 加入聊天室

* 邀請使用者

* 接受聊天邀請

三、好友私訊 (透過WebSocket實現聊天功能)
-------

* 好友搜尋

* 好友申請

* 建議可能認識的好友

* 接受好友申請
